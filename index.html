Documentación: Cambio de Imágenes y URLs

1. Cambiar Imágenes

Para modificar una imagen en el código HTML, simplemente cambia el valor del atributo src en la etiqueta <img>.

1.1 Cambiar el Icono de la App

Ubica el siguiente código en la sección <header>:

        <img src="https://www.appcreator24.com/srv/imgs/gen/2139191_ico.png" alt="Icono de App Creator 24">

Para cambiarlo, reemplaza la URL dentro del atributo src:

        <img src="URL_NUEVA_DEL_ICONO.png" alt="Nuevo Icono">

1.2 Cambiar Capturas de Pantalla

Dentro de la sección <section class="screenshots">, ubica los siguientes elementos:

        <div class="screenshot">
            <img src="http://files.corpsrtony.com/uploads/tonrut/3.jpg" alt="Captura 1">
        </div>

Cambia la URL en src para usar una nueva imagen:

        <img src="NUEVA_URL_DE_LA_IMAGEN.jpg" alt="Nueva Captura">

2. Cambiar URLs

Para cambiar los enlaces de descarga o de la página web, edita el valor del atributo href en la etiqueta <a>.

2.1 Cambiar el Enlace de Descarga

Ubica el botón de descarga dentro del encabezado:

        <a href="https://apk.e-droid.net/apk_t/app2139191_7409.apk?v=32" class="cta">Descargar para Android 📲</a>

Para cambiarlo, reemplaza la URL:

        <a href="NUEVA_URL_DE_DESCARGA.apk" class="cta">Descargar para Android 📲</a>

2.2 Cambiar el Enlace de la Página Web

Ubica el enlace de exploración:

        <a href="https://appcreator24herramientas.blogspot.com/" class="cta">Explorar Página Web 🌐</a>

Para cambiarlo, actualiza la URL dentro de href:

        <a href="NUEVA_URL_DE_LA_PAGINA" class="cta">Explorar Página Web 🌐</a>

Conclusión

Siguiendo estos pasos, puedes modificar fácilmente las imágenes y los enlaces en tu página HTML sin afectar su estructura.
